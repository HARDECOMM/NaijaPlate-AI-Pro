
# 🚗 NaijaPlate AI 

AI-powered Nigerian license plate detection and recognition system.

---

## 🔍 Overview

NaijaPlate AI Pro is a computer vision system designed to detect and recognize Nigerian vehicle license plates using a fine-tuned YOLOv8 model, OCR, and Google Gemini for enhanced accuracy.

---

## ❗ Problem

Manual vehicle logging is slow and error-prone. Generic OCR systems fail to accurately recognize Nigerian license plates due to unique formats and environmental challenges.

---

## ✅ Solution

- YOLOv8 (best.pt) for plate detection  
- OCR for text extraction  
- Gemini AI for correcting OCR errors  
- Prefix validation for Nigerian states and LGAs
